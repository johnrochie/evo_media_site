# Editing Evolution Media site content

To change text, prices, and other copy on the **Evolution Media** page (`/evomedia`), edit:

**`content/evomedia.json`**

- **hero** – Headlines, subheadline, button labels (See Our Work, Get started)
- **whatWeDo** – Section title and the 3 service cards (title, description, price). Icon keys: `briefcase`, `store`, `user`
- **whyUs** – Section title and 4 differentiators. Icon keys: `zap`, `bot`, `palette`, `trendingup`
- **howItWorks** – Section title and 3 steps. Icon keys: `messagecircle`, `hammer`, `rocket`
- **portfolio** – Section heading and subtitle only (project cards are still in code for now)
- **pricing** – Section title, custom CTA, and the Standard tier (name, subtitle, price, features, button text)
- **testimonials** – Section title and 3 testimonial cards (quote, author, role, avatar initials)
- **contact** – Section title, promise text, email, form dropdown options (website types, budget ranges), success message
- **footer** – Logo text, tagline, email, location, link labels, copyright, Privacy/Terms labels
- **nav** – Logo text and nav link labels

Save the file; the dev server will hot-reload. No need to touch component code for copy changes.
